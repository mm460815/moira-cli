# moira-cli
![![NPM version](https://img.shields.io/badge/moira-cli_v1.0.0-green
)](https://www.npmjs.com/package/moira-cli)
## 简介
- 基于vue3+vite4+ts开发的一个基础的脚手架工具

## 安装
```bash
 npm install moira-cli -g

 ```

## 📗使用
```bash
# 创建项目
moira create <project-name>
```
```bash
#  更新项目
moira update
```
```bash
# 查看脚手架版本

moira -v
moira -version
```
